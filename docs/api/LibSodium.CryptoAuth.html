Remarks

This class wraps the crypto_auth functions from libsodium's secret-key authentication API, which uses the HMAC-SHA-512-256 algorithm to generate and verify message authentication codes (MACs).

public static void GenerateKey(Span<byte> key)

Parameters

key Span<byte>

A writable buffer with a length of KeyLen bytes that will be filled with the generated key.

Remarks

This function uses a cryptographically secure random number generator to produce a secret key. Internally calls crypto_auth_keygen from libsodium.

Exceptions

ArgumentException

Thrown when the buffer length is not equal to KeyLen.
